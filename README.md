# dicesCLI
A game of dices for the command line

The

- Number of players
- Number of dices per roll
- Rolls per playes
- Dice width 

can be configured by changing the following constants:

// number of players
#define qtyPlayers  3

// number of dices
#define qtyDices        3

// dice width
#define diceWidth       9

// rolls per player 
#define rollsPerPlayer  1
